## Run locally

Instructions to setup project on your local machine.

1. Clone project
```
git clone https://beschtabrandon@bitbucket.org/gassomsfi/greyworm-epi.git
```
2. Install node packages
```
npm install
```
3. Run project
```
npm start
```
